差异化技术栈，只为更好体验

针对不同终端特性深度定制，而非简单的跨平台套壳。

桌面端 (Lite)

基于 C++17 & Win32 API 原生开发。无依赖单文件运行，KB 级内存占用。支持 Layered Window 透明磨砂渲染，完美融入桌面，不打扰是我的温柔。

移动端 (Pro)

使用 Flutter 构建，遵循 Material 3 设计规范。集成动态必应/GitHub 壁纸、后台自动更新、可视化数学分析报告，提供沉浸式交互体验。

双端协同

基于 Cloudflare D1 边缘数据库。无论在何处操作，数据毫秒级同步。采用乐观 UI 策略与冲突解决算法，支持断网操作与自动重试。
