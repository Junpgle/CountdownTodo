# CountDownTodo / Uni-Sync 隐私政策

**版本日期：2026年5月1日**
**生效日期：2026年5月1日**

CountDownTodo（及 Uni-Sync 协同套件，以下简称"本应用"）由 [Junpgle]（以下简称"我们"）开发并运营。我们深知个人信息对您的重要性，并会尽全力保护您的个人信息安全。本隐私政策旨在帮助您了解我们如何收集、使用、存储、传输、共享和保护您的个人信息。

**请您在使用本应用前，仔细阅读并充分理解本隐私政策。一旦您使用或继续使用本应用，即表示您同意我们按照本隐私政策处理您的个人信息。**

## 一、我们如何收集和使用您的个人信息

### 1.1 账号注册与登录

当您注册本应用账号时，我们会收集您的**手机号码或电子邮箱**用于创建账号和身份验证。我们采用自建JWT Token认证机制，您的密码将以加密形式存储在我们的服务器上。

### 1.2 核心业务数据

为提供待办事项管理、课程提醒、番茄钟、团队协作、跨设备同步、冲突处理等核心功能，我们会在本地或云端收集并存储以下数据：

- **待办事项数据**：标题、备注、开始时间、截止时间、全天状态、完成状态、删除状态、循环规则、提醒时间、分类/文件夹、团队归属、冲突状态等。
- **倒计时数据**：目标名称、目标时间、完成/删除状态等。
- **番茄钟数据**：专注记录、标签、统计、设置、跨端专注状态等。
- **课表数据**：课程名称、教师、教室、上课日期、节次、周次、备注等您导入或创建的信息。
- **屏幕使用时间**：各应用的使用时长统计（仅在您授权"使用情况访问权限"或配置 Windows TAI 数据源后处理）。
- **聊天与 AI 助手数据**：AI 待办助手的会话标题、消息历史、自定义提示词、模型名称、API 地址、深度思考开关等配置。
- **审计日志 (Audit Logs)**：为支持"版本历史回滚"与"冲突裁决"，我们会记录您对核心数据的修改历史快照（包含修改前后的字段内容）。

### 1.3 团队协作信息

当您参与团队协作（Uni-Sync 协同广场）时：
- **成员身份**：您的用户 ID、昵称及头像将对团队内其他成员可见。
- **数据共享**：您在团队频道内创建或更新的任务、日程将同步共享给所有团队成员。
- **管理通知**：管理员可实时查看团队公告的阅读确认情况。

### 1.4 局域网同步 (LAN Sync)

- 当您使用局域网同步功能时，您的设备会通过本地网络广播身份信息以发现其他设备。数据传输直接在您的设备间进行，**不经过我们的云端服务器**。
- 局域网同步可能处理待办、课程、倒计时、番茄钟等业务数据，具体范围取决于您开启的同步类型。

### 1.5 设备信息

为保障应用正常运行和提供版本更新服务，我们可能会收集以下设备信息：

- 设备型号、操作系统版本
- 应用版本号
- 设备标识符或本应用生成的设备 ID（用于数据同步、冲突处理、在线状态和版本更新）
- 网络请求信息（如 IP 地址、请求时间、接口路径、错误日志），用于保障服务安全和排查故障

### 1.6 多平台支持

本应用支持以下平台，功能可能因平台而异：

| 平台                  | 支持情况  | 特有功能                        |
|---------------------|-------|-----------------------------|
| **Android 手机端**     | 完整支持  | 桌面小部件、小米手环通信、屏幕时间统计、实时活动    |
| **Windows 电脑端**     | 完整支持  | 灵动岛/悬浮窗、多窗口、团队协作、全量分析       |

| **网页端**             | 较完整支持 | 通过 Cloudflare Zero Trust 访问，支持团队管理 |

本应用可能请求以下系统权限，您可以随时在系统设置中关闭：

### 1.8 AI 功能

当您使用"自然语言创建待办"、"图片识别待办"或"AI 待办助手"功能时：

- 您输入的文本、上传的图片、对话历史，以及为完成请求所需的待办、分类、课程、专注记录、冲突信息、团队信息等上下文，可能会发送至您配置的 AI 服务商进行处理。
- 默认 AI 接口为**智谱AI（BigModel）**；您也可以在设置中配置任意兼容 OpenAI API 格式的服务商、模型、API 地址和 API Key。
- AI 服务的 API Key、模型、API 地址、提示词和聊天历史默认保存在您的本地设备或本应用本地存储中；当您更换设备或清除应用数据时，相关配置可能需要重新设置。
- 我们不会主动将您的 AI 数据用于模型训练；但第三方 AI 服务商是否使用请求数据改进服务，取决于您选择的服务商及其隐私政策、服务条款和账户设置。
- 请勿在 AI 对话中输入身份证件、银行卡、精确住址、医疗健康、密码、密钥等敏感信息，除非您充分理解并接受相应风险。

### 1.9 壁纸服务

当您使用首页壁纸功能时：

- **必应每日一图 (Bing)**：我们会通过 `https://bing.biturl.top` 获取。
- **GitHub 随机壁纸**：访问特定开源仓库获取图片链接。

## 二、我们如何存储您的个人信息

### 2.1 存储方式

我们采用**本地 SQLite / SharedPreferences + 云端增量同步 (Oplog)** 混合方式。您的数据首先存储在本地，并在有网络且登录、开启同步或使用团队协作时，通过加密或非加密通道同步至服务器。

### 2.2 服务器架构说明

| 服务器 | 提供商 | 用途 | 传输协议 |
|--------|--------|------|----------|
| **阿里云服务器** | 阿里云（中国） | 核心同步、团队协作、审计日志、排行榜、WebSocket 实时同步 | **HTTP / WebSocket（未加密）** |
| **Cloudflare 服务器** | Cloudflare | 网页端访问、静态配置、版本更新、HTTPS 穿透 | HTTPS（加密） |
| **GitHub Raw** | GitHub | 静态资源配置 | HTTPS（加密） |

**特别安全风险告知：**
**阿里云服务器目前仍使用 HTTP / WebSocket 明文协议传输数据，存在被截获或篡改风险。** 我们正在积极推进备案与 HTTPS/WSS 改造工作。

### 2.3 存储地点

- 本地数据：存储在您的设备中
- 云端数据：存储于阿里云服务器（中国大陆）和 Cloudflare 相关服务
- 静态资源：部分配置信息存储于GitHub Raw

### 2.4 存储期限

- 账号数据：在您注销账号前持续保存
- 本地数据：在您卸载应用前持续保存
- 业务同步数据、团队数据和审计日志：在您删除、注销账号或团队管理员删除相关数据前持续保存；法律法规另有要求的除外
- 服务器访问日志和错误日志：通常保留不超过6个月
- AI 聊天历史和配置：保存在本地，直至您在应用内清除、卸载应用或清除应用数据

## 三、我们如何共享、转让、公开披露您的个人信息

### 3.1 共享

**我们不会主动将您的个人信息共享给第三方。** 以下情况除外：

- **智谱AI**：仅当您使用AI功能时，将您输入的文本或图片发送至智谱AI API进行处理
- **您自定义的 AI 服务商**：当您配置自定义 OpenAI 兼容接口时，将 AI 请求内容发送至您填写的 API 地址
- **小米穿戴SDK**：仅当您使用小米手环功能时，将待办、课表等同步数据发送至小米穿戴设备
- **Cloudflare**：用于网页端访问、HTTPS 代理、Zero Trust 访问和部分静态配置
- **GitHub / Bing 壁纸接口**：用于获取静态配置、隐私政策、版本信息或壁纸资源
- **法律法规要求**：根据法律法规、诉讼争议解决需要，或按行政、司法机关依法提出的要求

### 3.2 转让

我们不会将您的个人信息转让给任何公司、组织和个人，但以下情况除外：

- 事先获得您的明确同意
- 在涉及合并、收购或破产清算时，如涉及个人信息转让，我们会要求新的持有方继续受本政策约束

### 3.3 公开披露

我们仅会在以下情况下公开披露您的个人信息：

- 获得您的明确同意后
- 基于法律法规、法律程序、诉讼或政府主管部门强制性要求

## 四、我们如何保护您的个人信息

### 4.1 安全措施

我们已采取相应的安全防护措施保护您的个人信息：

- **数据加密存储**：敏感数据在服务器端加密存储
- **访问控制**：严格的服务器访问权限管理
- **HTTPS 加密传输**：Cloudflare 服务器和 GitHub Raw 使用 HTTPS/TLS 加密传输
- **Token 鉴权**：登录态使用 JWT Token 等机制进行身份验证
- **本地权限控制**：系统权限仅在相关功能需要时申请，您可以在系统设置中撤回

**重要安全提示：**

**阿里云服务器当前使用 HTTP / WebSocket 明文协议传输数据（未加密），存在数据被第三方截获或篡改的风险。** 我们正在推进服务器备案和 HTTPS/WSS 改造工作。在改造完成前，请您知悉并理解此风险。

### 4.2 安全事件处置

如发生个人信息安全事件，我们将按照法律法规要求，及时向您告知：

- 安全事件的基本情况和可能的影响
- 我们已采取或将要采取的处置措施
- 您可自主防范和降低风险的建议
- 对您的补救措施

您有权访问和更正您的个人信息：

- **账号信息**：在"设置"中查看和修改

### 5.2 删除

在以下情形下，您可以向我们提出删除个人信息的请求：

- 我们处理个人信息的行为违反法律法规
- 我们收集、使用您的个人信息，却未征得您的同意
- 我们处理个人信息的行为违反了与您的约定
- 您注销了账号
- 我们终止服务及运营

### 5.3 注销账号

您可以通过以下方式注销账号：

1. 打开应用，进入"设置" > "账号安全" > "注销账号"
2. 联系客服申请注销

账号注销后，我们将：

- 停止为您提供产品或服务
- 删除您的个人信息或进行匿名化处理
- 法律法规另有规定的除外

### 5.4 撤回同意

您可以通过以下方式撤回授权：

- **系统权限**：在设备系统设置中关闭相应权限
- **数据同步**：在应用设置中退出登录或关闭同步功能
- **AI 服务**：清除 AI 配置、删除 API Key 或停止使用 AI 助手功能

撤回同意不影响此前基于您授权已进行的个人信息处理活动的效力。

### 5.5 获取个人信息副本

您有权获取您的个人信息副本。如您需要，可通过客服联系我们获取。

## 十、定义

- **个人信息**：以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息，不包括匿名化处理后的信息
- **敏感个人信息**：一旦泄露或者非法使用，容易导致自然人的人格尊严受到侵害或者人身、财产安全受到危害的个人信息
- **匿名化**：通过对个人信息的技术处理，使得个人信息主体无法被识别，且处理后不能被复原的过程
- **设备信息**：包括设备型号、操作系统版本、设备标识符等

## 十一、特别提示

### 11.1 数据传输安全提示

**请您知悉：本应用阿里云服务器当前使用 HTTP / WebSocket 明文协议传输数据（未加密），可能存在数据被第三方截获或篡改的风险。** 我们正在积极推进服务器备案和 HTTPS/WSS 改造工作，完成后将启用加密传输。在此期间，建议您：

- 定期修改账号密码
- 如发现异常情况，请及时联系我们

### 11.2 平台功能差异

本应用在不同平台上的功能可能存在差异：

- **灵动岛/悬浮窗功能**：仅 Windows 电脑端可用

- **网页端**：支持通过 Cloudflare 服务器、阿里云服务器访问，功能可能受限
- **AI 待办助手**：依赖您配置的 AI 服务商接口，部分上下文会随请求发送给该服务商

### 11.3 服务器访问

- 手机端和电脑端可访问阿里云服务器和 Cloudflare 服务器
- 网页端通过 Cloudflare Zero Trust / HTTPS 入口访问后端能力
