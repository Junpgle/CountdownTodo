# assets/ — 静态资源目录

## 📌 目录构成

存放应用所需的图标、图片、启动页资源以及功能引导媒体文件。

---

## 📂 目录索引

| 目录 | 职责 |
|------|------|
| `icon/` | 应用图标（全平台） |
| `images/` | 内置壁纸、勋章图标、占位图 |
| `splash/` | 启动页静态图与动画资源 |
| `guide_media/` | 新功能引导中使用的演示视频或图片 |

---

## ⚠️ 使用规范

1. **资源注册**: 所有新添加的资源必须在根目录的 `pubspec.yaml` 中进行声明。
2. **图片优化**: 建议先进行无损/有损压缩再放入，单个文件尽量不要超过 500KB。
