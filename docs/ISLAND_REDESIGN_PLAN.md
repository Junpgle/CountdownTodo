# 灵动岛重新设计方案

## 📋 项目概述

本文档详细描述了灵动岛（Island UI）的重新设计方案，旨在简化界面、提升用户体验，同时保持所有现有功能的完整性。

---

## 🎯 设计目标

1. **界面简化**：去除复杂元素，让信息一目了然
2. **交互清晰**：只保留点击展开，去掉悬停展开
3. **视觉统一**：保持现有配色系统，优化视觉层次
4. **功能完整**：保留所有现有功能，不删减任何特性
5. **可扩展性**：支持后续添加系统控制功能

---

## 📐 核心设计原则

### 1. 紧凑模式只显示时间
- Idle状态下只显示"HH:MM"
- 不显示额外信息，保持极简

### 2. 点击展开，去掉悬停
- 移除所有悬停展开逻辑
- 统一使用点击交互

### 3. 三层视觉层次
- **小胶囊**：120×34, 100×46, 160×46
- **中卡片**：280×140, 260×130, 320×150/180
- **大卡片**：380×120（音乐播放器等）

### 4. 颜色系统
- 专注：紫色 (#6366F1)
- 提醒：橙色 (#FF9800)
- 完成：绿色 (#4CAF50)
- 放弃/危险：红色 (#D32F2F)
- 背景：深色 (#1C1C1E)

## 🔔 强提醒机制设计

### 提醒强度分级

| 级别 | 视觉效果 | 行为 | 适用场景 |
|------|----------|------|----------|
| **Light** | 普通显示 | 自动消失 | 复制链接、普通通知 |
| **Medium** | 显示+轻微动画 | 可自动消失 | 待办/课程即将开始 |
| **Strong** | 脉冲+发光+展开 | 必须确认 | 番茄钟结束、倒计时结束 |

### Strong级别视觉效果

#### 脉冲动画
- 缩放：1.0 → 1.15 → 1.0，循环3次
- 颜色：紫色 ↔ 橙色渐变
- 边框：发光效果（半径20px）

#### 自动展开
- 脉冲结束后1秒自动展开为卡片
- 显示明确的操作选项
- 5秒无操作重复脉冲提醒

### 番茄钟结束强提醒

**Step 1：倒计时归零瞬间**
```
┌─────────────────┐
│     00:00       │  ← 脉冲动画（3次）
└─────────────────┘
```

**Step 2：自动展开卡片**
```
┌────────────────────────────────────┐
│   🍅 番茄钟结束！                   │
│   专注事项 | 已完成 25分钟           │
│                                    │
│   ┌────────────┐  ┌────────────┐   │
│   │  继续计时  │  │    完成    │   │
│   └────────────┘  └────────────┘   │
└────────────────────────────────────┘
```

**Step 3：无操作重复提醒**
- 5秒无操作 → 再次脉冲
- 10秒无操作 → 再次脉冲
- 直到用户操作

### 正计时结束处理
- 正计时结束不提醒
- 用户主动点击结束时才触发确认流程

## 🔄 状态回退机制

### 状态分类

| 类型 | 状态 | 回退行为 |
|------|------|----------|
| **基础状态** | Idle, Focusing | 栈底，不可pop |
| **展开状态** | StackedCard | 点击空白回到基础状态 |
| **确认状态** | FinishConfirm, AbandonConfirm | 必须用户选择，受保护 |
| **临时状态** | CopiedLink, FinishFinal | 自动消失或点击消失 |
| **提醒状态** | ReminderPopup, ReminderCapsule | 用户处理后消失 |
| **系统控制** | QuickSettings, MusicPlayer等 | 点击空白回到Idle |

### 栈操作规则

```dart
// 入栈：临时状态
stack.push(IslandState.copiedLink, data: payload);

// 出栈：恢复下层
final restored = stack.pop(IslandState.copiedLink);

// 替换栈顶：reminderSplit 替换 focusing
stack.replaceTop(IslandState.reminderSplit, data: payload);

// 替换栈底：idle <-> focusing 切换
stack.replaceBase(IslandState.focusing, data: payload);

// 清空回 idle
stack.clearToIdle();
```

### 受保护状态

```dart
static const protectedStates = {
  IslandState.finishConfirm,   // 完成确认
  IslandState.abandonConfirm,  // 放弃确认
  IslandState.finishFinal,     // 完成最终
  IslandState.copiedLink,      // 复制链接
  IslandState.reminderPopup,   // 提醒弹窗
};
```

外部payload无法覆盖受保护状态。

## 🔧 技术实现要点

### 1. 移除悬停逻辑

需要删除的代码：
- `_onHoverEnter()` 方法
- `_onHoverExit()` 方法
- `_hoverDebounce` 定时器
- `_minStayTimer` 定时器
- `_isHovered` 状态变量
- `_canShrink` 状态变量
- `MouseRegion` 的 `onEnter` 和 `onExit` 回调

### 2. 简化状态机

保留的核心状态：
- `idle` - 时钟
- `focusing` - 专注中
- `stackedCard` - 详情卡片
- `finishConfirm` - 完成确认
- `abandonConfirm` - 放弃确认
- `finishFinal` - 完成最终
- `reminderPopup` - 提醒弹窗
- `reminderSplit` - 双胶囊提醒
- `reminderCapsule` - 提醒胶囊
- `copiedLink` - 复制链接

可考虑移除的状态：
- `hoverWide` - 悬停展开（改用点击展开）
- `splitAlert` - 分屏通知（功能可合并）

### 3. 双层灵动岛实现

```dart
Widget _buildCopiedLink() {
  return Column(
    mainAxisSize: MainAxisSize.min,
    children: [
      // 主岛：保持当前状态
      _buildMainIsland(),
      SizedBox(height: 4),
      // 副岛：链接提示
      _buildLinkSubIsland(),
    ],
  );
}
```

### 4. 脉冲动画实现

```dart
// 缩放脉冲
AnimationController _pulseController = AnimationController(
  duration: Duration(milliseconds: 500),
  vsync: this,
);

Animation<double> _pulseAnimation = TweenSequence([
  TweenSequenceItem(tween: Tween(begin: 1.0, end: 1.15), weight: 25),
  TweenSequenceItem(tween: Tween(begin: 1.15, end: 1.0), weight: 25),
  TweenSequenceItem(tween: Tween(begin: 1.0, end: 1.15), weight: 25),
  TweenSequenceItem(tween: Tween(begin: 1.15, end: 1.0), weight: 25),
]).animate(_pulseController);

// 循环3次
_pulseController.repeat(count: 3);
```

### 5. 颜色闪烁实现
